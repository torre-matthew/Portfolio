About Me

Born and raised in the United States Virgin Islands, I now live and work in Raleigh, North Carolina. Since graduating from NC State University, I've worked as a project manager, scrum master, product owner and product manager. I don't only talk software though, I also love building it. I write complex web apps on Node.js. using various javascript libraries, database technologies , and the occasional CSS framework.
